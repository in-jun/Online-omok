게임 안내

다양한 플레이어들과 실시간으로 오목을 즐기며 새로운 전략을 습득하세요.

주요 특징

  • 실시간 플레이: 다른 플레이어들과 실시간으로 오목 대전을 즐길 수 있습니다.
  • 간편한 접속: 복잡한 절차 없이 간편한 방법으로 게임에 참여할 수 있습니다.
  • 자동 매칭: 서버에 접속하면 자동으로 매칭되어 상대 플레이어와 빠르게 대전할 수 있습니다.

사용 방법

  1. 페이지로 이동: 먼저 여러분의 웹 브라우저를 열고 페이지로 이동해주세요.
  2. 서버 연결 중: 페이지에 도착하면 "서버 연결중" 메시지가 화면에 표시될 것입니다. 잠시만 기다려주세요.
  3. 유저 매칭 중: 서버 연결이 완료되면 "유저 매칭중" 메시지가 뜨게 됩니다. 이제 다른 플레이어와의 매칭을 기다려봐요.
  4. 게임 시작: 매칭이 성공하면 오목판에 돌을 놓아 게임을 시작하세요. 각 플레이어는 돌의 색으로 구분되어 턴에 맞춰 돌을 놓을 수 있어요.
  5. 시간을 잘 활용: 주어진 시간 내에 돌을 놓지 않으면 시간초과 입니다. 빠르고 정확하게 결정하세요!

게임 목표

게임의 목표는 가로, 세로, 혹은 대각선으로 다섯 개의 돌을 연속으로 놓아 상대방을 제압하는 것입니다.

각 플레이어는 번갈아가며 돌을 놓을 수 있으며, 이어진 돌 다섯 개가 만들어지면 해당 플레이어의 승리입니다. 게임은 15x15 크기의 오목판에서 펼쳐지며, 상대방을 막아가며 승리에 도전하세요!

소스 코드 확인

"온라인오목"의 소스 코드는 GitHub 레포지토리에서 확인할 수 있습니다. 소스 코드를 통해 게임의 동작 원리를 이해하고, 기여할 수 있는 기회도 열려 있습니다.
